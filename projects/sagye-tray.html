Sagye Joinery Lacquer Tray

전통 사개물림의 결구 구조를 현대적인 리빙 오브젝트로 재해석한 옻칠 트레이 프로젝트입니다. 단순한 수납 기능을 넘어, 사용자의 손길과 생활 공간 안에서 고유한 물성을 드러내는 사물을 목표로 했습니다.

Category
Living Object / Tray / Material Study
Period
2026.02 – 2026.04
Work
Personal Project

Sketch / Blueprint / Structure

형태와 결구 구조의 전개

스케치에서 시작해 각 부품의 비례와 단면을 정리하고, 사개물림의 교차 구조를 트레이의 지지 방식으로 확장했습니다.

Binding / Top view / Making

조립과 표면 마감

맞물리는 프레임을 손으로 조립하고 상판을 결합했습니다. 완성된 구조 위에는 검은 옻칠을 반복해 올려 깊이 있는 표면을 만들었습니다.

Object in space

옻칠의 깊은 표면과 결구의 그림자가 일상의 작은 장면을 조용하게 구분합니다.
